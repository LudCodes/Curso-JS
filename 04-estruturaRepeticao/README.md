# Módulo 04 — Estruturas de Repetição

Neste módulo estudo como fazer o JavaScript repetir tarefas automaticamente, evitando código repetido e criando lógicas mais inteligentes.

---

## O que foi estudado
- O que são loops (laços de repetição)
- Estrutura `for`
- Estrutura `while`
- Estrutura `do...while`
- `for...of`
- `for...in`

---

## Prática
- Exercícios na pasta `/exercicios`
- Projeto prático na pasta `/projetos`

---

## O que aprendi
- Fazer o JavaScript repetir tarefas
- Controlar início, condição e parada de loops
- Evitar código repetido
- Criar lógicas mais eficientes
