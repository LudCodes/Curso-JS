# Fundamentos da Linguagem

Neste módulo, estudo os fundamentos da linguagem JavaScript, entendendo como os dados são armazenados, manipulados e comparados.

---

## O que foi estudado
- Tipos de dados primitivos
- `var`, `let` e `const`
- Operadores aritméticos
- Operadores de comparação
- Operadores lógicos
- Coerção de tipos (implícita e explícita)

---

## Prática
- Exercícios na pasta `/exercicios`
- Projeto prático na pasta `/projetos`

---

## O que aprendi
- Declarar variáveis corretamente
- Identificar e utilizar tipos de dados primitivos
- Utilizar operadores aritméticos, lógicos e de comparação
- Entender coerção de tipos
